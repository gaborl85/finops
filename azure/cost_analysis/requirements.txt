# Azure Cost Analysis Module - Requirements
# This module uses PowerShell scripts with Azure CLI and Azure Cost CLI.
# No Python dependencies required.

## Required Tools

### 1. Azure CLI (az)
- Installation: https://docs.microsoft.com/cli/azure/install-azure-cli
- Required for: Authentication and subscription management
- Verify: `az --version`

### 2. Azure Cost CLI
- Installation: `dotnet tool install -g azure-cost-cli`
- Source: https://github.com/mivano/azure-cost-cli
- Required for: Querying cost data from Azure Cost Management
- Verify: `azure-cost --version`

### 3. PowerShell
- Minimum: PowerShell 5.1
- Recommended: PowerShell 7+ (cross-platform)
- Windows: Built into Windows 10/11
- macOS/Linux: https://github.com/PowerShell/PowerShell/releases

## Permissions Required

- Azure Cost Management Reader role on subscriptions
- Azure Reader role for subscription access

## Verification Commands

```powershell
# Check Azure CLI
az --version

# Check Azure Cost CLI
azure-cost --version

# Check PowerShell version
$PSVersionTable.PSVersion

# Verify authentication
az account list --output table

# Verify cost access
az costmanagement query --help
```
